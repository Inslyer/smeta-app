# Инструкция: как пользоваться Генрихом

## Один раз: подготовка

1. Открой https://daniila-smeta.streamlit.app
2. В **боковой панели** проверь, что внизу написано «База цен: Postgres @ ep-wispy-star…». Если SQLite — значит секрет не дошёл, напиши Юрию.
3. Если нет ни одного поставщика, в сайдбаре «➕ Добавить»:
   - Название: `ЭТМ`
   - Тип: «Материалы»
   - Политика цен: «Переменные» (для ЭТМ) или «Фиксированные»

## Каждый раз: новый проект

### Шаг 1. Создать проект
Вкладка **«📦 Счета поставщиков»** → «➕ Новый проект» → введи название (например, «Объект Х»).

### Шаг 2. Загрузить все счета поставщиков
На той же вкладке выбери поставщика (общий для всей партии) → перетащи **все PDF/XLSX-счета** этого поставщика сразу. Генрих парсит каждый, показывает превью. Если всё ок — «💾 Сохранить все в проект».
Повтори для каждого поставщика.

### Шаг 3. Загрузить смету Заказчика и подрядчика
Вкладка **«📤 Загрузка смет»** → два поля. Перетащи XLSX в каждое.

### Шаг 4. AI-сопоставление
Вкладка **«📊 Сводная смета»**:
1. «✨ Сопоставить позиции» — Генрих ищет пары работ. Светофор: 🟢🟡🟠⚪.
2. «🧪 Классифицировать остаток» (если есть неприматченные у подрядчика) — Генрих говорит какие из них компоненты, а какие — отдельные доп. работы вне сметы Заказчика.
3. Выбери Проект → «🧮 Подобрать цены закупки» — Генрих сопоставляет материалы Заказчика с позициями счетов.

### Шаг 5. Скачать
«📥 Скачать сводную смету» внизу. Excel с пятью+ листами: Сводная, Аналитика, Исходники, Счета.

## Что должно получиться в Excel
- В строках материалов в колонке «Поставщик» написано имя ЭТМ + № счёта + дата (не tmpXXXX).
- Маржа по разделу, где нет цен подрядчика/поставщика, показана как `—`, а не 100%.
- В «Аналитике» материалы попадают в правильную категорию (кабель в «Кабель и проводка», даже если в названии есть слово «лотки»).
- Доп. работы подрядчика «вне сметы» уменьшают общую маржу проекта.

## Если что-то не так
- В сайдбаре «🩺 Диагностика окружения» — должно быть `✓ задан` напротив `ANTHROPIC_API_KEY` и `DATABASE_URL`.
- «🧹 Сбросить кеш сессии» — выбрасывает все распарсенные данные из памяти этой вкладки. Полезно если результат застрял в старом матче.
- В сайдбаре «Версия:» показывает git-хеш — должен совпадать с последним коммитом в `Inslyer/smeta-app`. Если старый — Streamlit Cloud залип на старой сборке, открой Manage app → Reboot.
