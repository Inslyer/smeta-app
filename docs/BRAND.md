# Брендинг ГРАФТИО в Генрихе

## Имя продукта
Только **«Генрих»**. Не «Генрих от Графтио», не «Графтио Генрих». ГРАФТИО — материнский бренд, отображается только через логотип. В UI и Excel-документах ГРАФТИО не упоминается текстом.

## Никогда не использовать в UI
- Слово «Claude» (даже в спинерах) — заменяется на «Генрих». Исключение: имя модели `claude-sonnet-4-6` в env-переменной и техническая диагностика — оставлять как есть.
- Слово «клиент» в смыслах «заказчик сметы». Используется **«Заказчик»**. В моделях и коде ОК — `client_estimate`, `client_idx`.

## Палитра
```
primary lime     #B1EC52   акценты, hover, подчёркивание активной вкладки
bright green     #7BBE2F   главный зелёный кнопок
yellow accent    #F4E821   правая часть градиента, hover
charcoal text    #2D2D2D   текст-черный на светлом
bg dark          #0E0E0E   главный фон (тёмная тема)
secondary bg     #1A1A1A   sidebar, expander, metric chip
text on dark     #F2F2F2   почти-белый текст
muted on dark    #A0A0A0   caption / placeholders
```

## Шрифт
Inter (Google Fonts), веса 400 / 500 / 600 / 700 / 800. Заголовки `font-weight: 800`, `letter-spacing: -0.5px`.

## Логотип
Реальный логотип ГРАФТИО — буква «О» из слова «ГРАФТИО», стилизованная под планету: разорванное угольное (или белое на тёмном) кольцо + диагональная зелёно-жёлтая полоса градиента `#45B649 → #D7DF23`.

- `assets/graftio_symbol.svg` — только знак, viewBox `720 10 160 155`. Используется в шапке (56×56) и сайдбаре (40×40).
- `assets/graftio_planet.svg` — полный горизонтальный логотип «ГРАФТИО PARTNER IN ELECTROMOBILITY», скачан с graftio.ru.

## Тон коммуникации
Спокойный, профессиональный, без перебарщивания с эмодзи. Допустимы аккуратные эмодзи в названиях вкладок (📤📊📦) и кнопок (💾🧮🧪🗑). Спинеры в первом лице от Генриха: «Генрих читает счёт», «Генрих сопоставляет позиции», «Генрих разбирает остаток».

## Тёмная тема
Дефолтная. Задана через `.streamlit/config.toml` (`base = "dark"`) + кастомный CSS в `app_with_ai.py` (форсирует тёмный фон у компонентов которые тема не покрывает: dataframe, file_uploader, inputs).

## Запрещено
- Светлая тема (фон #FFFFFF на содержимом).
- Использовать буквенный знак «Sk» / любые посторонние моноограммы, которые случайно встречались в pptx-презентации.
- Замещать SVG-знак эмодзи 🪐 в активной странице (только как fallback).
