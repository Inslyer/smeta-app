# Сводная смета — автоматизация закупки

Приложение для закупщика: загрузить смету клиента и смету подрядчика, получить сводную смету с маржой по позициям и разделам.

## Стэк

- Python + Streamlit
- Claude API для сопоставления позиций
- SQLite для хранения базы цен поставщиков

## Деплой

Streamlit Community Cloud — главный файл `app.py`.

В **Secrets** надо положить:
```
ANTHROPIC_API_KEY = "sk-ant-..."
CLAUDE_MODEL = "claude-sonnet-4-5"
```
